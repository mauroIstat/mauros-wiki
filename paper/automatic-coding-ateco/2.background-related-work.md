# Background and Related Work

## 2.1 Automatic coding in Official Statistics

Automatic coding is not a new problem in Official Statistics. National Statistical Institutes have long developed methods and tools to support the assignment of official codes from textual descriptions in domains such as economic activities, occupations, and product classifications. This reflects a recurrent operational need to reduce the burden of manual coding while preserving consistency and control in the use of statistical classifications.

Early work on automatic coding in Official Statistics was largely based on deterministic and lexical approaches. A well-known example is ACTR, which provided an important reference for later developments in statistical coding systems [ACTR]. In the Italian case, this line of work led to the development of CIRCE, a rule-based system designed to assign classification codes from textual descriptions [CIRCE]. These systems have played an important role in statistical practice because they offer transparency, operational control, and methodological stability, especially when coding decisions must remain traceable and consistent over time.

At the same time, the effectiveness of these systems depends heavily on the availability and continuous maintenance of dictionaries, normalization procedures, synonym lists, and matching rules [CIRCE]. This creates a structural burden, especially when classifications are revised or when textual descriptions vary substantially from the institutional terminology used in the taxonomy. As a result, rule-based systems may remain effective in controlled settings, but they are less flexible when coding tasks involve paraphrases, ambiguous formulations, or evolving language.

## 2.2 Embedding-based semantic retrieval

Sentence-level embedding models provide a different way to approach this problem. Instead of relying mainly on lexical overlap, these methods represent textual descriptions in a semantic vector space, where similarity reflects broader meaning rather than exact word matching [SBERT]. This makes them particularly relevant for coding tasks in which user queries and official class descriptors may refer to the same activity using different wording.

In this setting, semantic retrieval differs from direct classification. Rather than predicting a single label through a fixed decision rule, the system compares a query representation with representations of the target categories and returns the most similar candidates. This ranking logic is especially relevant when the output is intended to support coding through a shortlist of plausible labels, rather than through a single fully automatic assignment [SemanticATECO2026] [LLMCoding2025].

An important implication of this approach is that performance depends not only on the embedding model itself, but also on how the target classification is represented in the semantic space. A category may be associated with one text, with several related texts, or with a single representation obtained by combining multiple texts. For this reason, the design of label representations becomes part of the methodological problem rather than a secondary implementation detail [SemanticATECO2026].

## 2.3 Challenges in real-world coding

These methodological developments must be considered in the context of real-world coding tasks. In practice, textual descriptions are often short and ambiguous. They may also reflect the language used by respondents, or other users rather than the institutional terminology of the official classification. This creates a persistent mismatch between user language and taxonomy language, which makes coding difficult even when the classification itself is well defined.

Moreover, coding in OS should not be evaluated only in terms of predictive accuracy. An equally important question is whether a prediction can be considered sufficiently reliable for its intended use. This raises methodological issues related to assessing the uncertainty of coding methods, e.g., confidence thresholds, prediction sets, and conformal prediction.

Finally, the difficulty of coding is not uniform across the label space. Some categories or groups are more easily identified than others, and model performance may vary substantially across divisions depending on linguistic overlap, semantic ambiguity, and class-specific characteristics. For this reason, robustness across subgroups is an important part of the evaluation problem, rather than a secondary descriptive issue.

This literature suggests that both automatic coding and semantic retrieval are relevant to the present study, but it also points to a more specific gap. While there is substantial work on traditional coding systems and increasing interest in embedding-based methods, less attention has been paid in the literature to framing economic-activity coding as a ranking problem over a structured classification and assessing it within a broader methodological framework.
