# Experimental Design

## Section Goal

Make the comparison logic explicit: which models, which representation strategies, which dataset combinations, and how preferred configurations are selected.

## What This Section Must Specify

## Recommended Selection Criteria

The paper should avoid choosing the preferred model only because it has the highest raw score. A defensible rule should combine:

- shortlist quality at 5 digits
- robustness at 2 digits
- stability across divisions
- computational feasibility for operational deployment
