# Data sources

The empirical setting of this study is based on two complementary data sources. The first is the CP2021 occupational classification and an auxiliary resource used to enrich the official class descriptions with 6-digit occupational entries. The second is a labelled dataset of Italian free-text occupation descriptions paired with gold-standard CP2021 codes. In the benchmark design, the classification resources define the target occupational categories and their official textual descriptions, while the labelled dataset is used both to evaluate retrieval performance and to enrich the textual representation of the 5-digit codes.

## CP2021 classification and 6-digit enrichment resource

CP2021 is the occupational classification used in this study as the target coding framework. The classification is organized hierarchically and includes titles and descriptions at multiple levels of granularity. In this study, occupation coding is performed at the 5-digit level, since the labelled dataset contains 5-digit CP2021 codes and statistical production requires occupation data at this level of detail.

Table 1 summarizes the hierarchical structure of CP2021 in the source dataset.

| Level | Sheet | Number of codes |
| --- | --- | ---: |
| 1-digit | `primo_digit` | 9 |
| 2-digit | `secondo_digit` | 40 |
| 3-digit | `terzo_digit` | 130 |
| 4-digit | `quarto_digit` | 510 |
| 5-digit | `quinto_digit` | 813 |

**Table 1.** Hierarchical levels of CP2021 and number of codes at each level.

The official CP2021 resource used in the study contains 1,502 entries across the hierarchical levels included in the source dataset.

To enrich the official classification, the study also uses an auxiliary data source containing a more detailed occupational inventory linked to CP2021. The resource includes 6,813 complete rows associated with 6-digit occupational entries and is used to augment each 5-digit class with a set of more concrete occupational denominations that instantiate the broader class.

This enrichment resource is methodologically important because it provides lexical and semantic detail that is not always available in the official 5-digit description. While the official classification captures the institutional definition of the occupational class, the 6-digit entries offer a finer-grained representation of the occupations that fall under that class. In the retrieval setting adopted here, these 6-digit labels act as semantically informative examples that can improve the alignment between user queries and the textual representations of occupational codes.

In the benchmark, the enriched resource is not treated as a separate prediction target. Rather, it is used to strengthen the textual representation of the 5-digit CP2021 classes. The 6-digit occupational entries are aggregated under their parent 5-digit code and incorporated into the text associated with each code, producing a richer retrieval corpus while leaving the evaluation target unchanged at the 5-digit level.

## Labelled occupation dataset

The second core data component is a labelled dataset of Italian occupation descriptions paired with gold-standard CP2021 codes assigned through prior annotation.

The labelled dataset contains 61,881 records and serves as the benchmark of the study. Each record is interpreted as a realistic occupational description that the model must map to the appropriate CP2021 class.

The dataset plays a dual role in the experimental design.

### Benchmark role

First, it is the benchmark dataset used to evaluate retrieval performance. Each annotated occupation description is converted into a query, and the gold-standard CP2021 code is used to determine whether the correct occupational category is retrieved, within the top \(k\), or not retrieved at all.

This use of the dataset provides a direct evaluation setting: the benchmark measures how effectively multilingual embedding models can retrieve the correct CP2021 occupational category from naturally occurring free-text descriptions.

### Descriptor-enrichment role

Second, the same labelled dataset is used to enrich the informational content associated with each 5-digit code. To do so, a TF-IDF-based procedure is applied at the 5-digit level to extract the n-grams that are most characteristic of the query texts associated with each occupational class. These terms are then added to the textual representation of the corresponding code.

This step introduces a weakly supervised form of enrichment. No external labels are added beyond the annotated dataset itself; instead, lexical patterns observed in the query corpus are used to bring the representation of each code closer to the vocabulary actually used in occupation descriptions. The impact of this enrichment on retrieval performance is then assessed in the benchmark.

The use of the annotated dataset for both benchmarking and lexical enrichment is therefore not redundant, but structurally coherent with the study design. In the benchmark role, the dataset provides the empirical distribution of occupation descriptions to be matched. In the enrichment role, it provides class-specific linguistic cues that can make the textual representation of the target codes more compatible with that distribution.

## Complementarity of the two data sources

Taken together, the two data components reflect the two sides of the retrieval problem. The CP2021 resources define the institutional label space and the textual information associated with that space. The labelled dataset defines the observed query space and the gold-standard mapping between queries and occupational classes.

This separation is analytically useful because it allows the study to test a precise hypothesis: retrieval quality depends not only on the embedding model, but also on the degree of semantic compatibility between these two textual spaces. The official classification provides the institutional starting point, the 6-digit resource adds occupational detail, and the labelled dataset contributes both the benchmark itself and the lexical evidence needed to approximate the language of real occupation descriptions.
